# arc.store — Premium E-commerce Frontend

A production-quality, zero-framework e-commerce storefront built with modern Vanilla JavaScript and CSS.

## 🚀 Key Features
- **Modular ES6:** Clean architecture with functional imports/exports.
- **Micro-interactions:** Custom cart drawer, product quick-view, and heart animations.
- **Dynamic System:** Responsive design with CSS variables for easy theming.
- **Performance:** Lightweight and lightning-fast with zero external dependencies.

## 🛠️ Tech
- Vanilla JavaScript (ES6+)
- Modern CSS (Grid & Flexbox)
- Semantic HTML5
